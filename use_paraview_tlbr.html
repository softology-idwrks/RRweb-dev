The RapidReader ParagraphView Toolbar

Please "mouse over" the items on the toolbar below for information

Above you can see the paragraphView with embedded Notes, Highlited text and Dog-Ear markup. You can click on the Note marker to access them individually. You can also use the Viewer function (menubar View>Viewers) to skip through a document based on your markup.

RapidReader is more than a speed reading tool. With it you can capture PDF's, Web Pages, MS Word docs and more. Then you can add your own markup directly in RapidReader.

It's easy to save web pages for off-line reading, add comments to PDF's, excerpt text from "white papers" you can even save an entire log of e-mails as a single RapidReader file. Then add Notes, Highlite text "Dog-Ear" a page for easy reference

Once you open a document or web page in RapidReader you can add your own markup for review or sharing. Simply save as a RapidReader file to preserve the document plus your Notes, Highlites and Dog-Ears.

Using the ParagraphView tool bar you can...

  • Create a Note and attach it to a word location
  • Export all your Notes as a separate file for quick review.
  • Highlite text for emphasis (in TurboView Highlites are bright green)
  • Export any text you Highite for quotes or excerpts.
  • Use the Viewers to skip through a document using your markup
  • E-mail a RapidReader (r2d) file with markup as an attachment.

Progress Bar
Shows your progress through your document by percentage completed and position by page number.
Speed Indicator
Shows current speed setting when you resume in TurboView
Speed Icon
Calls the Set Speed Dialog. You can set the speed between 100 and 950 words per minute.
Create a Note
Select a word. Click on the Note Icon to launch the note pad. The note location will be marked with a number e.g. {N001}
Dog-Ear a "page"
Select a word. Click on the Dog-Ear Icon to place a marker at the selected word. The Dog-Ear location will be marked with a number e.g. {Dog001}
Highlite Text
Click on the Highlite Icon to change to Highlite Mode. Click and drag across text to Highlite. When finished Click on the Highlite Icon again to return to Normal Mode.
Save as .r2d file
Calls the Save Dialog. You can save any document as a RapidReader File. This will copy the entire document in .r2d format and keep your markup intact.
Return To App

This allows you to return to the program from which the document was loaded into RapidReader. For instance when you call RapidReader from within MS Word.

Very useful for editing the original when proofreading in RapidReader.
Help Files
Help files include full product information , troubleshooting, a quick start "How Do I?" quide and access to the Tutorial.
