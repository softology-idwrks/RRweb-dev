General Instructions for Installing and Activating RapidReader Standard Edition:

The Installer Package you downloaded from the RapidReader site is contained in a .zip file.

Unzip the package and inside you'll find:

  • RapidReader Standard 6.0 Installer
  • The Local Validator Utility for activating RapidReader
  • Installation Instructions
  • The RapidReader Tutorial
  • A quick start "Cheat Sheet" with "how to" information for opening web pages and documents in RapidReader.
  • A "Power User" Tips and Tricks guide for using RapidReader's markup features to save time on lots of information tasks
  • Some examples of books and other content for RapidReading

Installing RapidReader for Windows

Right Click on the RapidReader Download Folder.

1) Select Extract All to your desktop or other convenient location.

This folder contains the installer, the valiator utility, user information and sample files

In the folder you will see the Installer Icon and a folder labeled Validator.

2) Double Click on the RapidReader Installer Icon and follow the prompts to install.

3) Then EXIT RapidReader

You should now have a RapidReader Icon on your desktop

4) In the RapidReader Download Folder Open the Local Validator Folder.

 

 

You will see a number of files in the folder.

5) Double Click on the item indicated with the RapidReader Book Icon and labeled RR6LocalValidator.exe

You will receive a validation authentication dialog box

6) Launch RapidReader by Double Clicking on the RapidReader icon on your desktop

Once RapidReader opens you should now see the words Normal Mode and Full Mode in the lower right hand corner of the RapidReader Window.

 

Please Note:


If you receive an "error 5: access denied" error message or if RR remains in "DEMO MODE" please see instructions here.

If the installation "hangs" while "registering file types" exit RapidReader and re launch.

If you are still not able to validate please contact support.

There is a brief tutorial available upon install and in the RapidReader Download Folder. We highly recommend that you spend the few minutes to experience what reading 500 wpm feels like as well as get an overview of features and functions to get you up and running quickly.

To see what this technique is all about please visit "what is flooding?"

If you don't see the RapidReader icon on the MS Word, Internet Explorer or Outlook* toolbar...

You may need to manually enable the RapidReader add-ins.


Please Note: Check this first. in IE 8+ and newer MS Office Products the tool bar has been changed and the RR icon may be hidden. If so you will have to look for it on the toolbar under: Tools>manage add-ons>enable-disable add-ons. You will see RR as one of the choices.

Please click here for excellent general information about enabling and managing Microsoft Add-Ins.


To do this you will use the RapidReader app integration utility which was installed with RapidReader. This utility is what puts the button on the supported applications.

Please follow the instructions below.

*requires RR PRO

    To Manually Enable Add-ins...


  1. Close all MS Office components
  2. From your Start Menu or Computer C drive
  3. Go to Programs>Softology>rapidreader app-integration
  4. Click on App Integration Utility.

    5. This will launch a dialog box.
    You will then be asked to enable add-ins for Word, Explorer and Outlook.

  5. Click enable for each one.



Programs>Softology>RapidReader App-Integration

Click on App Integration utility

After you've enabled the addins, when you open Microsoft Word, Explorer or Outlook the RapidReader Button should be visible on the tool bar.


Installing RapidReader on Windows Vista and Windows 7+:

When installing RapidReader on your Vista or Windows 7+ system some additional steps may be required.


If you receive an error message indicating "access is denied" it simply means that you need to reset your administrator permissions to allow the installation. Please follow the steps below.

Please Exit RapidReader then

1) Go to the Start menu and CLICK on COMPUTER

2) Click on the "C" DRIVE

3) Click on PROGRAMS

4) Open Program Files\Softology\RapidReader6.
RIGHT CLICK on RapidReader6.exe

5) In the Right Click menu click on RUN AS ADMINISTRATOR

6) When the “Unknown program…” dialog box pops up, choose the “ALLOW” option.

Then re launch RapidReader and continue with the normal license authentication process.

After authentication you should see the words "FULL MODE" in the lower right hand corner of the RapidReader Window.
